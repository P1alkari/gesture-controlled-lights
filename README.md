
# Gesture Controlled Lights (Python + Arduino)

Control 5 LEDs on an Arduino using finger gestures detected from a webcam.
The system uses MediaPipe Hands in Python and sends the finger count (0–5) to an Arduino over serial.

## Hardware
- Arduino UNO
- 5x LEDs
- 5x 220Ω resistors
- Breadboard + jumper wires

LED pins: D2, D3, D4, D5, D6
